# 🌤️ Weather App

A modern and responsive Weather Application built using React. It provides real-time weather updates with a clean UI and smooth user experience.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time weather data (API integration)
- 🎨 Dynamic background based on weather conditions
- 🕒 Recent search history
- 📱 Fully responsive (mobile + desktop)
- ⚡ Smooth loading experience (no flicker)

---

## 🛠️ Tech Stack

- React (Vite)
- CSS (Glassmorphism UI)
- OpenWeather API

----

git clone https://github.com/saoud786/weather-app.git
